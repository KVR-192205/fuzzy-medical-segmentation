# X-ray Image Segmentation using ANFIS

## Overview
This project uses ANFIS (Adaptive Neuro-Fuzzy Inference System) to segment chest X-ray images.

## Steps
1. Read X-ray image
2. Convert to grayscale
3. Train ANFIS using pixel intensity
4. Classify pixels into regions

## Output
- Segmented X-ray image
- Regions: dark, medium, bright

## Tools
- MATLAB
- Fuzzy Logic Toolbox
